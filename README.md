 3D Roll-a-Ball Project

Team Members: Aida Aimurzayeva and Asset Ayanat
Chosen Project
3D Roll-a-Ball

Project Description
This project is based on the official Unity Roll-a-Ball tutorial.  
The player controls a rolling ball, collects pickups, avoids an enemy, and tries to complete the game objective.
The project includes player movement, camera follow system, collectibles, score tracking, enemy AI interaction, win and lose conditions, UI systems, and additional custom gameplay features.

How to Run the Project
1. Open Unity Hub
2. Add the project folder
3. Open the project in Unity 6

Playable Build
Run the executable (.exe) file from the Build folder.

Controls
- WASD / Arrow Keys — Move player
- ESC — Open / Close pause menu
- Resume Button — Continue the game
- Restart Button — Restart the level
